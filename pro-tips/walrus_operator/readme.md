# 🦭 The Python Walrus Operator (:=)

A quick demonstration of the **Assignment Expression** (affectionately known as the "Walrus Operator"), introduced in Python 3.8. 

This repository contains a simple script showcasing how to use `:=` to write cleaner, more concise Python code by assigning and returning a value in a single line.

## 💡 What does it do?

Traditionally, if you want to assign a variable and then check its value or print it, you need multiple lines of code:

```python
# The Old Way (2 lines)
happy = True
print(happy)
