## 📂 Current Snippets

### 1. The Walrus Operator (`walrus_operator.py`)
Introduced in Python 3.8, the assignment expression operator `:=` (affectionately called the walrus operator) allows you to assign and return a value in the exact same expression. 

This snippet demonstrates how to use it to:
* Drastically shorten `while` loops.
* Clean up conditional `if` statements.
* Print and assign variables in a single line.
