# 💎 Python Diamond Pattern Generator

A simple but logic-heavy Python script that generates an ASCII diamond shape in the terminal based on user input. 

Unlike basic pattern scripts, this program dynamically calculates the necessary blank spaces and asterisks for **both odd and even numbers**, ensuring a perfectly symmetrical diamond every time.

## 🧠 Concepts Demonstrated
* **Control Flow:** `while` loops for continuous user prompts and `break` conditions.
* **Conditional Logic:** Using the modulo operator (`%`) to split routing for odd vs. even inputs.
* **Iteration:** `for` loops combined with `range(start, stop, step)` to sequence the rows.
* **String Manipulation:** Utilizing Python's string multiplication (`" " * space_count`) for clean formatting.

Run the script in your terminal:
