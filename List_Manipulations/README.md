# 📋 Python List Manipulations

Welcome to the List Manipulations module! This section is dedicated to mastering one of Python's most versatile and frequently used data structures: the List. 

Lists are the backbone of data handling in Python. This collection of scripts and exercises covers everything from basic indexing to advanced functional programming techniques.

## 🎯 What's Covered in this Module?

This directory contains practice files and mini-projects that explore the following core concepts:

### 1. The Basics (Creation & Access)
* Creating lists (empty, populated, mixed data types).
* Accessing elements using positive and negative indexing.
* **Slicing:** Extracting sub-lists using `[start:stop:step]`.

### 2. Modifying Data (Mutating Lists)
* Adding elements: `.append()`, `.insert()`, and `.extend()`.
* Removing elements: `.remove()`, `.pop()`, and `del` keyword.
* Clearing and copying lists.

### 3. Organizing & Searching
* Sorting data: `.sort()` (in-place) vs `sorted()` (new list).
* Reversing list order: `.reverse()` vs `[::-1]`.
* Finding elements: `.index()` and checking existence with the `in` operator.
* Counting occurrences with `.count()`.

### 4. Advanced Techniques 🚀
* **List Comprehensions:** Writing elegant, one-line loops to generate lists.
* **Functional Methods:** Using `map()`, `filter()`, and `lambda` functions with lists.
* **Multi-dimensional Lists:** Navigating and manipulating matrices (lists within lists).
