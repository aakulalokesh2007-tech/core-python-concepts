# 🐍 Core Python Concepts

A dedicated workspace for practicing and mastering the fundamental concepts of Python. This repository contains a collection of scripts, exercises, and mini-projects that I have built to strengthen my understanding of backend logic, data manipulation, and software design.

## 🎯 Purpose
The goal of this repository is to track my progress and serve as a quick-reference library for Python syntax, algorithms, and design patterns. It moves from basic syntax to more advanced integrations with external libraries and databases.

## 🗂️ Repository Structure
*(Note: As you upload files, organize them into folders like this to keep it clean!)*

* `📁 basics/` - Variables, loops, conditionals, and functions.
* `📁 data_structures/` - Lists, dictionaries, tuples, and sets.
* `📁 oop/` - Object-Oriented Programming (Classes, inheritance, polymorphism).
* `📁 file_handling/` - Reading and writing text, CSV, and JSON files.
* `📁 mini_projects/` - Small, self-contained applications applying multiple concepts.

## 🛠️ Technologies & Libraries Explored
While focusing on pure Python, these scripts occasionally touch upon libraries I use in larger projects:
* **Built-in modules:** `os`, `time`, `pickle`
* **Data Manipulation:** `pandas`
* **Databases:** `mysql.connector`
* **GUI Development:** `tkinter`
